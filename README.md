# STEM Career Classroom Guide

A collaborative hub for teachers to share resources, lesson plans, and templates for integrating STEM careers into classroom curriculum.

## Purpose

This repository aims to support educators in connecting classroom learning with real-world STEM careers. By sharing standardized templates, lesson ideas, and career spotlights, we can help students understand the diverse opportunities in STEM fields and the pathways to get there.

## How to Contribute

We welcome contributions from educators at all levels! Here's how you can help grow this resource:

1. **Browse Issues**: Check the [Issues](https://github.com/kippy-bing/stem-career-classroom-guide/issues) tab for suggested improvements or new content requests.
2. **Create an Issue**: If you have an idea for a new template, lesson plan, or career profile, open an issue to describe it.
3. **Fork & Branch**: Fork this repository and create a new branch for your changes.
4. **Add Content**: Follow existing templates and guidelines to add your contribution.
5. **Submit a Pull Request**: Open a pull request to propose your changes. Reference the related issue in your PR description (e.g., `Closes #1`).
6. **Review & Merge**: After review, your contribution will be merged into the main repository.

## Getting Started

Explore the `templates/` directory for ready-to-use templates, such as the **STEM Career Spotlight** template. Use these as starting points for creating your own resources.
